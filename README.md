Projektbeschreibung

Passwort-Validator - Ein Tool zur Validierung von Passwörtern basierend auf einer definierten Passwort-Policy. Das Tool stellt sicher, dass Passwörter sicher sind und nicht in einer Liste häufiger oder schwacher Passwörter enthalten sind.

Funktionalität:

Prüft, ob ein Passwort den grundlegenden Sicherheitsanforderungen entspricht.
Unterstützt die Überprüfung von Passwörtern mit folgenden Kriterien:

-Mindestens 8 Zeichen.
-Mindestens eine Ziffer (0–9).
-Enthält sowohl Groß- als auch Kleinbuchstaben.
-Kein häufiges oder schwaches Passwort.
-Mindestens ein Sonderzeichen (!#$%()*+,-./:;=?@[]^_{|}~).
